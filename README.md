## What is DevMingler?

DevMingler is a cozy build club for developers, tinkerers, learners, and anyone experimenting with code. Our Discord community is active again, growing steadily, and making space for project channels, co-working energy, small wins, and open-source experiments.

## What we do

- Share projects, prototypes, and work-in-progress ideas.
- Co-work, ask questions, and help each other get unstuck.
- Celebrate small wins, learning notes, and shipped experiments.
- Build beginner-friendly open-source tools and demos.
- Make space for curious people at every experience level.

## Featured project: Footprints

Footprints is a reflective tooling experiment for exploring ChatGPT conversation history and turning it into grounded, useful reports.

| Project | What it is |
| --- | --- |
| [footprints-codex-skill](https://github.com/DevMingler/footprints-codex-skill) | A Codex skill for creating evidence-grounded conversation-history reflections. |
| [footprints-custom-gpt](https://github.com/DevMingler/footprints-custom-gpt) | A Custom GPT companion for the Footprints workflow. |
| [footprints-bts](https://github.com/DevMingler/footprints-bts) | Behind-the-scenes notes, experiments, and supporting material. |
